<h4>Mysql Komutları </h4>
Temel seviyede aşağıdaki gibi sayılabilir.<br> 
Ama daha kapsamlı uygulamalar için bunların parametrelerini ve diğer komutları kullanmak gerekir.<br>
CREATE DATABASE: Yeni bir veritabanı üretir<br>
CREATE TABLE: Yeni bir tablo üretir<br>
INSERT INTO: Veritabanına yeni kayıt ekler<br>
SELECT FROM: Veritabanından bilgi okumaya yarar<br>
WHERE: Yapılan işlemin etkileyeceği sütunları belirtir<br>
ORDER BY: Seçim işleminde sıralama veya tersten sıralama yapar<br>
UPDATE SET: Veritabanında değişiklik yapar<br>
DELETE FROM: Bilgi siler<br>
Yukarıdakiler temel SQL komutlarıdır ve neredeyse her türlü veritabanında geçerlidir. Aşağıdaki komutlar ise özel durumlarda kullanılabilir.<br>
SHOW TABLE: Tabloların listesini verir<br>
DESCRIBE: Tablo sütunlarının adını yazar<br>
DROP DATABASE: Veritabanını toptan siler<br>
DROP TABLE: Tabloyu toptan siler (Dikkat edin sadece içini boşaltmaz. tablo tarih olur, yanlış kullanmayın)<br>
TRUNCATE TABLE: Tablodaki bilgileri siler, Tablo ayarlarını sıfırlar<br>

ALTER DATABASE: Veri tabanının öz niteliklerini değiştirir.<br>
ALTER TABLE: Tabloya alan ekleme, tablodaki alanı düzenleme ve silme işlemlerini yapar.<br>
ALTER VIEW: Görünüm değiştirmede kullanılır.<br>
CREATE DATABASE:Yeni bir veritabanı oluşturur.<br>
CREATE INDEX:Yeni bir index oluşturur.<br>
CREATE TABLE: Yebi bir tablo oluşturur.<br>
DELETE: belirtilen tablodan yeri WHERE deyimi ile saptanan satırı siler.<br>
DROP DATABASE: belirtilen veritabanını siler.<br>
DROP INDEX:belirtilen index i siler.<br>
DROP TABLE:belirtilen tabloyu siler.<br>
DROP USER:belirtilen kullanıcıyı siler ama kullanıcının sahip olduğu ; tablo, database ,vsvs silinmez.<br>
INSERT:tabloya yeni bir satır ekler.<br>
RENAME TABLE: var olan tablonun adını değiştirir.<br>
SELECT: veri tabanındaki tablonun tamamını yada WHERE ile belirtilen alanı gösterir.<br>
